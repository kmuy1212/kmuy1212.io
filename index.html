Introduccion Web

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para crear contenido web. Es la piedra angular de la creación de páginas web y permite definir la estructura y el contenido de una página web. HTML utiliza etiquetas (tags) para definir diferentes elementos en una página web, como encabezados, párrafos, imágenes, formularios y enlaces. Las etiquetas se utilizan para indicar al navegador cómo presentar el contenido en la página web. HTML es un lenguaje de marcado simple y fácil de aprender, y es compatible con la mayoría de los navegadores web modernos. Con HTML, puedes crear páginas web desde cero o editar páginas existentes para mejorar su estructura y contenido. En resumen, HTML es una herramienta fundamental para cualquier desarrollador web, ya que permite crear páginas web eficientes y bien estructuradas que se pueden mostrar en cualquier navegador web.

HTML

Componentes Principales

  • Etiqueta HTML

    indica al navegador que el documento es una página web y especifica la versión de HTML que se está utilizando.
  • Encabezado de la página (head)

    contiene información importante sobre la página, como el título de la página, la descripción, las palabras clave y el enlace a la hoja de estilos CSS. No se muestra en la página web.
  • Cuerpo de la página (body)

    contiene el contenido principal de la página, como texto, imágenes, videos, enlaces y otros elementos. Todo lo que se ve en la página web se encuentra en el cuerpo.
  • Encabezados (headers)

    se utilizan para dividir el contenido de la página en secciones y proporcionar una estructura jerárquica. Los encabezados van desde h1 (el más importante) hasta h6 (el menos importante).
  • Párrafos (paragraphs)

    se utilizan para escribir texto y dividir el contenido de la página en bloques de texto.

CSS

Funciones Principales

  • Font-family

    Define la familia tipográfica. Tenemos 3 opciones para definir la elección de la familia tipográfica:

  • Font-size

    Define el tamaño de la fuente y el valor se puede escribir en pixels o rems.

  • Color y background-color

    Color define el color de la tipografía y background-color, el color de fondo de un elemento. Los colores se pueden escribir de varias formas

  • Width

    efine el ancho de un elemento, el valor se puede escribir en pixels, rems o porcentaje. Las etiquetas de bloque tienen por defecto un width de 100%.

  • Max-width o min-width

    Definen el ancho máximo o mínimo de un elemento. Muy importante para adaptar el sitio a los diferentes tamaños de pantalla

  • Padding

    Es la distancia desde el borde de un elemento hasta su contenido. Esta propiedad es imprescindible si la caja tiene color de fondo o borde

  • Margin

    Es la distancia entre un elemento y otro (desde el borde de un elemento hacia afuera). Muchas etiquetas HTML tienen margin por defecto por ejemplo: el body, los encabezados, los párrafos y las listas.
  • Border

    Define el borde de un elemento, su color, su estilo y grosor.
